Help

  1. Pinch or spread screen to zoom in or out of the map.
  2. Drag map from one point in the opposing direction to pan across the map.
  3. Click on Road Map to change map type to a vector map showing road network and building structure.
  4. Click on Hybrid map to change map type to the default satelite view of the campus.
